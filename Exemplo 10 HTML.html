Exemplo 10 HTML - Tabelas

As tabelas em HTML são representadas pela tag <table>, que são úteis para exibir dados tabulares, tais como informaçãoes em linhas e colunas.

Descrição dos principais atributos para criação de tabelas em HTML

  • <table>: Esta é a tag princípal que envolve toda a tabela HTML
  • <tr>: Esta tag representa uma linha da tabela e nela teremos as tags <th>, <td>
  • <th>: Esta tag representa o cabeçalho da tabela. th é o acrônimo de header.
  • <td>: A tag <td> (table data) representa a célula da tabelas
  • <theader>: Tag semântica que representa o cabeçalho. Ela é utilizada juntament com a tag <th>
  • <body>: Tag semântica que representa o corpo da tabela, nela geralmente usamos a tag td
  • <tfoot>: Tag semântica que representa o radapé da tabela.
