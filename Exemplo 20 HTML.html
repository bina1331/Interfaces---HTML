Visão geral sobre CSS

CSS é um acrônimo para folhas de estilo em cascata. Essa técnica visa personalizar, estilizar um documento HTML. Para isso, usamos regas CSS que representam uma estrutura que define como os elementos selecionados devem ser estilizados. Essa regra é composta pelas seguintes partes:

  • Seletor indica quais elementos serão afetados pelo estilo definido.
  • Bloco de Declaração é o bloco composto por propriedades e valores, esse bloco será aplicado a um elemento HTML.

Tipos de Seletores

Em CSS, seletores são padrões usados para selecionar e estilizar os elementos HTML em uma página web. Eles definem quis elementos receberão as regras de estilo que serão declaradas dentro das chaves. Possuimos vários tipos de seletores, os principais são:

Seletores de ID

O seletor de ID é aquele que aplica a regra CSS a um componente HTML que tenha um atributo ID no componente html. Esse seletor de id deve ser aplicado a um único elemento html, haja vista que o ID deve ser único. Este seletor tem prioridade sobre os demais seletores(de classe e de tipo).

Este seletor sempre começará com um "#", seguido pelo nome do seletor e suas propriedades e valores.
