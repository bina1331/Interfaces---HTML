Exemplo 13 HTML - Formulários

os formulários HTML são uma das principais formas de interagir com aplicação web. Esses formulários utilizam diversos componentes que melhoraram a experiência do usuário.

Para trabalhar com formulários, devemos usar a tag <>em conjunto com outras tags que vão representar os componetes de formulários, tais como: labels(rótulos), inputs(caixa de texto), botões, elementos de checagem, de seleção entre componentes. Neste exemplo, demonstraremos o uso dos métodos Get e Post

Métodos de Protocolo

Get

Quando trabalhamos com a criação de formulários HTML, nós deveremos na maior parte das vezes utilizar o protocolo de rede HTTP, esse protocolo possui vários métodos que são utilizados para facilitar o processo de comunicação entre o cliente e o servidor

O método GET é aquele que anexa os dados do formulário a uma URL, esses dados do formulário são visíveis na barra de endereço do navegador.

É importante saber que esse método não é recomendado para operações de inclusão, pois os dados vão aparecer para qualquer pessoa, o que pode ser inseguro.

Post

Já o método POST é aquele que envia os dados para o servidor no corpo da página. Essas informações não são vísiveis para o cliente sendo um pouco mais seguras que quando utilizamos o método GET.
