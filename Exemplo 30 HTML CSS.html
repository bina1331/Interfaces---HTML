Exemplo 30 - FLEXBOX

O FlexBox é um modelo de layout em CSS, que foi projetado para facilitar a criação de layouts complexos. Ele é espcialmente útil para a criação de layouts responsivos.

Principais Proriedades do FlexBox

Flex-Direction

a propriedade flex-direction permite alterar a direção na qual os elementos HTML serão exibidos, a seguir demonstramos o seu uso.
