Exemplo 7 HTML - Listas Ordenadas <ol> e Listas Não Ordenadas <ul>

Este exemplo mostra como criar listas ordenadas e não ordenadas, incluindo subitens (listas aninhadas).

Listas ordenadas <ol>

As listas ordenadas são aquelas que seguem uma ordem númerica ou por letras. São utilizadas para mostras conteúdo html em formato de listas.

Exemplo da lista ordenada <ol> juntamente com os elementos dessa lista que é representado pela tag <li>

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4
  5. Item 5

Exemplo 2 de listas ordenadas

  1. Item 1
    1. subitem 1.1
    2. subitem 1.2
    3. subitem 1.3
    4. subitem 1.4
    5. subitem 1.5
  2. Item 2
  3. Item 3
  4. Item 4
    1. subitem 4.1
    2. subitem 4.2
    3. subitem 4.3
    4. subitem 4.4
    5. subitem 4.5
  5. Item 5

Listas Não Ordenadas - <ul>

As listas não ordenadas são aquelas que representam os elementos da lista por meio de bullets. Essas listas são muito utilizadas na criação de menus. Abaixo demonstramos alguns exemplos de listas não ordenada

Exemplo de lista não ordenada

  • Item 1
  • Item 2
    • Item 2.1
    • Item 2.2
    • Item 2.3
    • Item 2.4
      • 2.3.1
      • 2.3.2
      • 2.3.3
  • Item 3
  • Item 4
  • Item 5
